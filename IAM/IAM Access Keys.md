#Amazon 


[[IAM Users]] can have 0, 1 or 2 active or inactive access keys that are made of 2 parts: 

1. Access Key ID
2. Secret Access Key 

The keys can be activated and reactivated. 

> Use different profiles on the [[AWS CLI]] for each user.
