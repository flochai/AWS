#Amazon 

Managed DNS of Amazon that provides 2 main services: 

1. register domains
2. host zones 

This is a global service that is replicated between regions so it is globally resilient. 


## 1. Register Domains

Create a zonefile
Create managed NS
Add NS records to the zonefile of top level domains
