#General 


1. Block Storage
	 files separated in equal sized pieces
	 example: [[EBS - Elastic Bloc Store]]
	 \- not easy to share / local
	 \+ performance
	 bootable
	 mountable


2. Object Storage
     data + metadata + key = object 
     no file system
     top layer
     not bootable
     not mountable
     [[S3]]


4. File Storage 
      **shared** file folders
      [[EFS - Elastic File System]] for example
      mountable
	  not bootable
