#Amazon 

IOPS = 1MB

### 1. ST1: Throughput Optimized

- cheap
- fast
- 125GB - 16TB
- Good for Sequential DATA 
- 40MB/s per TB
- Burst: 250MB/s per TB


### 2. SC1: Cold HDD

- cheapest EBS storage
- Infrequent workloads
- 250 IOPS max 
- 250MB/s throughput
- 12MB/s per TB
- Burst: 80MB/s per TB
